Sobre o Sistema

Sistema Integrado de Gestão de Ordens de Serviço desenvolvido para a Coordenação de Suporte e Logística - Manutenção. Permite criar, gerenciar e acompanhar ordens de serviço de forma eficiente e organizada com sistema de fila inteligente por prioridade.

Principais Funcionalidades

  • Sistema de fila inteligente por prioridade
  • Criação e gestão de ordens de serviço
  • Controle de status e prioridades
  • Dashboard com análises e gráficos
  • Histórico de serviços concluídos
  • Upload e gestão de fotos
  • Assinaturas digitais
  • Relatórios fotográficos
  • Personalização de interface

Tecnologias Utilizadas

HTML5 CSS3 JavaScript Tailwind CSS Canvas API
